# 📱 Mobile Sales Analysis Project

## 📊 Overview
This project focuses on analyzing mobile sales data using Power BI. It provides meaningful insights into sales trends, top-performing brands, and revenue patterns.

## 🛠 Tools Used
- Microsoft Excel
- Power BI

## 📁 Files Included
- Mobile Sales Data.xlsx → Raw dataset
- Mobile Sales Project.pbix → Interactive dashboard

## 📈 Key Insights
- Top selling mobile brands
- Monthly revenue trends
- Customer purchasing behavior
- Region-wise sales performance

## 🎯 Objective
To transform raw sales data into interactive and visual insights for better business decision-making.
